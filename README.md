# Отчёт о тестировании приложения Бонусной системы

## Краткое описание

06.04.2021 - 06.04.2021 было проведено Функфиональное тестирование приложения для валидации номеров карт.

На тестирование затрачено: <1 час>

В результате тестирования выявлены следующие дефекты:
* [Не верный подсчет суммы при исчислении бонуса](https://github.com/gavrilowwa/java122/issues/1)

## Описание процесса тестирования

В процессе тестирования использовались следующие артефакты:
*  баг-репорты в Github Issue


В качестве тестовых данных использовались данные:
* 0.3 Валидное значение
* 0.6 Валидное значение

Тестирование производилось в следующем окружении:
* Windows 10 x64
* Java version "11.0.10"
